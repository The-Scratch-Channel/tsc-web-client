# Reporting a bug

To report a bug, you'll need a GitHub account.

> [!IMPORTANT]
> If this bug is a security vulnerability, go to [Security Vulnerability Reporting](../security/README.md)

Click [here](https://github.com/The-Scratch-Channel/tsc-web-client/issues/new/choose) to make a new issue. Make sure you give plenty of detail about your OS, browser and how to recreate the issue.
